# 🇧🇷 Candidato360: Transparência Eleitoral e Prestação de Contas

## Sobre o Projeto
O **Candidato360** é um aplicativo Android desenvolvido para dar mais **transparência** nas eleições e acompanhar a vida pública no Brasil. 

Nosso objetivo é empoderar o cidadão, fornecendo acesso fácil a:
* **Transparência de Gastos:** Acompanhe em tempo real as despesas de campanha dos candidatos.
* **Fiscalização Pós-Eleição:** Monitore o desempenho e as ações do candidato eleito durante seu mandato.
